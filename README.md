
# HR Analytics Dashboard (Power BI)

# Project Overview
This project is an HR Analytics dashboard built using Power BI. 
The main objective is to analyze employee data and understand attrition trends, workforce distribution, and key HR metrics.

# Objectives
 Analyze employee attrition (why employees leave)
 Identify high-risk departments
 Understand employee demographics
 Improve decision-making using data insights

#  Key Insights
 Overall attrition rate of the company
 Department-wise attrition analysis
 Gender-wise employee distribution
 Age group analysis
 Job role and salary impact on attrition
 Monthly attrition trends
